• MedDiary
  • A medical state tracker

  • Description:
  • MedDiary is a mobile appliction written using Xamarin that takes standard data such as "Temperature", percentage data such as "Intensity of headache", or boolean data such as "Dry mouth". This data is stored and read into a graph showing the progression of the subject's illness, making the perscribing of medicine easier for doctors.

  • Supported platforms:
  • Android (SDK 16 to latest)
