• TeaChess
  • A chess problem loading tool

  • Description:
  • TeaChess is a cross-platform desktop appliction intended to make writing and loading chess positions simple and fast. It uses a new format, which only stores the current game state, and not every move which lead to the current board position.

  • Supported platforms:
  • Linux (X11)
