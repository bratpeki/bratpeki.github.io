• EnthApp
  • A chemistry class assistant

  • Description:
  • EnthApp is a GUI application based on Python and the Tkinter framework. It features an enthalpy, chemical balance, molar mass and orbital layout calculator, as well as a WIP periodic table. Besides that, tables providing information about enthalpy and entropy of certain atoms and compounds are available.
