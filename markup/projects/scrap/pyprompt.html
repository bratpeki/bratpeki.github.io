• PyPrompt
  • A modular Python Shell

  • Description:
  • PyPrompt is a text-based application serving a modular top-level terminal layer, meaning that you can add custom commands as well as run commands available in lower levels (bash, zsh, cmd...). The project is in need of a revamp.

  • All major desktop platforms
